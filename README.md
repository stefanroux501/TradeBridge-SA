E-Commerce Marketplace | PHP, Bootstrap, SQL
Built a responsive, full-stack C2C e-commerce website to manage online transactions allowing users to create an account to buy and items and register as a seller to sell items on the website to other users.
The website has RBAC and admin panel to verify sellers, release funds, ban users and remove products that was listed
Engineered the backend architecture using PHP to handle user authentication and secure data routing.
Designed the frontend with Bootstrap to ensure a smooth, cross-device user experience.
